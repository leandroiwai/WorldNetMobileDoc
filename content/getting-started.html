Get Started

First of all we need to create an Android project in Android Studio. Open your Android Studio and select the option "Start a new Android Studio project" in the Quick Start panel.

Step 1 - Create a project in Android Studio.

Step 2 - Fill the fields with the project name and the company domain.

Step 3 - Select API 16 (Android 4.1 - Jelly Bean) or above.

Step 4 - Fill the fields with the project name and the company domain.

Step 5 - In this example we will use the default options.


<uses-permission android:name="android.permission.INTERNET"/>
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE"/>
<uses-permission android:name="android.permission.ACCESS_NETWORK_STATE"/>

Step 6 - Go to the AndroidManifest.xml and set these permissions.

Step 7 - Go to the MainActivity and try do implement the interface CoreAPIListener. If it appears, we are ready to go.
